# 栈&队列部分

#### 1.有效的括号

[LeetCode题目地址](https://leetcode-cn.com/problems/valid-parentheses/)

```JavaScript
var isValid = function (s) {
    const map = new Map()
    map.set('[', ']')
    map.set('{', '}')
    map.set('(', ')')

    const stack = [];

    for (let i = 0; i < s.length; i++) {
        if (map.has(s[i])) {
            stack.push(map.get(s[i]))
        } else {
            if (stack.pop() !== s[i]) {
                return false
            }
        }
    }
    if (stack.length !== 0) {
        return false
    }
    return true
};
```
